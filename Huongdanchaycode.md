Hướng dẫn chạy code

Bước 1: Tải mã nguồn về máy
Tại trang GitHub này, nhấn vào nút Code (màu xanh).

Chọn Download ZIP.

Sau khi tải về, hãy giải nén file vào một thư mục trên máy tính.

Bước 2: Mở dự án bằng Visual Studio Code
Mở phần mềm VS Code.

Chọn File > Open Folder... (hoặc nhấn Ctrl + K Ctrl + O).

Tìm đến thư mục bạn vừa giải nén và nhấn Select Folder (giải nén thêm file images.zip).

Bước 3: Cài đặt Extension hỗ trợ (Quan trọng)
Để đảm bảo các liên kết CSS và Javascript hoạt động chính xác nhất, bạn nên sử dụng Live Server:

Nhấn vào biểu tượng Extensions trên thanh công cụ bên trái (hoặc Ctrl + Shift + X).

Tìm kiếm từ khóa Live Server.

Nhấn Install để cài đặt.

Bước 4: Chạy Website
Ở cột danh sách file bên trái, nhấn chuột vào file Home.html để mở mã nguồn.

Nhìn xuống góc dưới cùng bên phải của cửa sổ VS Code, nhấn vào nút Go Live.

Trang web sẽ tự động hiển thị trên trình duyệt mặc định của bạn.
